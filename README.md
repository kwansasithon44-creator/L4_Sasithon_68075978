# L4_Sasithon_68075978
จัดทำโดย นางสาวศศิธร เสาร์ประโคน
โครงงานนี้เป็นการพัฒนาเว็บแอปพลิเคชันรูปแบบ Image Gallery
โดยใช้ภาษา HTML, CSS และ JavaScript เพื่อแสดงรูปภาพในลักษณะตาราง (Grid Layout)

หน้าเว็บจะแสดงรูปภาพอาหารหลายรายการ เมื่อผู้ใช้คลิกที่รูปภาพใด รูปภาพนั้นจะแสดงแบบขยายใหญ่ขึ้นในรูปแบบ Modal เพื่อให้ผู้ใช้สามารถดูรายละเอียดของภาพได้ชัดเจนมากขึ้น

การออกแบบหน้าเว็บใช้ CSS Grid เพื่อจัดวางรูปภาพให้แสดงผลอย่างเป็นระเบียบและรองรับการแสดงผลบนหน้าจอหลายขนาด (Responsive Design)
ส่วนการทำงานของการเปิด–ปิดภาพขนาดใหญ่ใช้ JavaScript ในการควบคุมการแสดงผลของ Modal

โครงงานนี้มีวัตถุประสงค์เพื่อฝึกทักษะการพัฒนาเว็บแอปพลิเคชันพื้นฐาน การจัดการโครงสร้างไฟล์ และการทำงานร่วมกันของ HTML, CSS และ JavaScript
